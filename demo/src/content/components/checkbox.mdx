# Checkbox

`c2n-checkbox` provides the same functionality as a native `<input type="checkbox">` enhanced with styling and animations.

### Installation

```bash tag=CodeBlock
npm install @c2n/checkbox
```

### Import

```typescript tag=CodeBlock
import '@c2n/checkbox'
```
