# List Item

`c2-list-item` is a basic ui block that can be slotted into various components including list, dropdown, menu, etc.

### Installation

```bash tag=CodeBlock
npm install @c2n/list-item
```

### Import

```typescript tag=CodeBlock
import '@c2n/list-item'
```

<br />
## Examples
<br />
---

```html tag=MdxCodeBlock,compack=true
<c2-list-item data-style="width:100px; height: 100px; --test-v: 10px">Default list item</c2-list-item>
```

---

```html tag=MdxCodeBlock,compack=true
<c2-list-item selected>Selected list item</c2-list-item>
```

---

```html tag=MdxCodeBlock,compack=true
<c2-list-item selected id="abc" class="custom-selected-state">

</c2-list-item>

    --c2-list-item-selected-bg-color: red;
  }

```
