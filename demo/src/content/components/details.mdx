# Details

`c2-details` represents data inform of details/summary. Behind the scene, it uses native details/summary html tag.

### Installation

```bash tag=CodeBlock
npm install @c2n/details
```

### Import

```typescript tag=CodeBlock
import '@c2n/details'
```
