# Expansion Panel

`c2-expansion-panel` is an variant of `c2-details` which does not ues _details/summary_ native component.

### Installation

```bash tag=CodeBlock
npm install @c2n/expansion-panel
```

### Import

```typescript tag=CodeBlock
import '@c2n/expansion-panel'
```
