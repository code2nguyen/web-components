# List Item

`c2-list-item` is a basic ui block that can be slotted into various components including list, dropdown, menu, etc.

### Installation

```bash tag=CodeBlock
npm install @c2n/list-item
```

### Import

```typescript tag=CodeBlock
import '@c2n/list-item'
```

<br />
## Examples
<br />
---

```html tag=MdxCodeBlock compack
<c2-list-item>Default list item</c2-list-item>
```

---

```html tag=MdxCodeBlock compack
<c2-list-item selected>Selected list item</c2-list-item>
```

---

```html tag=MdxCodeBlock compack
<c2-list-item selected class="c2-list-item-selected-state-1">Selected list item 1</c2-list-item>
<style>
  .c2-list-item-selected-state-1 {
    --c2-list-item-selected-bg-color: transparent;
  }
</style>
```

## API

| Name                                   | Type         | Default                 | Description                                                                          |
| -------------------------------------- | ------------ | ----------------------- | ------------------------------------------------------------------------------------ |
| # Attribute                            |              |                         |                                                                                      |
| selected                               | .bool        | `false`                 | If `true`, the component is selected                                                 |
| disabled                               | .bool        | `false`                 | If `true`, the component is disabled                                                 |
| value                                  | .string      | `''`                    | The `value` dispatch on selecting item                                               |
| # Events                               |              |                         |                                                                                      |
| selected-change                        | .CustomEvent | N/A                     | The `detail` attribute contains `value` if item is selected otherwhise it is `false` |
