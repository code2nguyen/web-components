# Color Slider

`c2-color-slider` allows users to select hue of a color from 0 to 360.

### Installation

```bash tag=CodeBlock
npm install @c2n/color-slider
```

### Import

```typescript tag=CodeBlock
import '@c2n/color-slider'
```
