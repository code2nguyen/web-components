# Select

`c2-select` component composed a `c2-list` and a button will trigger display a list of options. 

### Installation

```bash tag=CodeBlock
npm install @c2n/select
```

### Import

```typescript tag=CodeBlock
import '@c2n/select'
```

<br />
## Examples
<br />
---

```html tag=MdxCodeBlock,compack=true,component=c2-select
<c2-select client:only="lit" placeholder="Select your country">
  <c2-list-item value="US">United States</c2-list-item>
  <c2-list-item value="CA">Canada</c2-list-item>
  <c2-list-item value="FR">France</c2-list-item>
  <c2-list-item value="CO">Colombia</c2-list-item>
</c2-select>
```
