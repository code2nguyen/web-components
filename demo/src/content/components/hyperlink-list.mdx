# Hyperlink List

`c2-hyperlink-list` is a container accepts a list of hyperlink tag, then style them in form of a vertical list.

### Installation

```bash tag=CodeBlock
npm install @c2n/list-item
```

### Import

```typescript tag=CodeBlock
import '@c2n/list-item'
```

<br />
## Examples
<br />
---

```html tag=MdxCodeBlock,compack=true
<c2-hyperlink-list selected-url-pattern="/link3">
  <a href="/link1">Link 1</a>
  <a href="/link2">Link 2</a>
  <a href="/link3">Link 3</a>
  <a href="/link4">Link 4</a>
</c2-hyperlink-list>
```
