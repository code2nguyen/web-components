# Overlay

`c2-overlay` component is designed to work with Popover API of browser. Used to represent information in form of Popup.

### Installation

```bash tag=CodeBlock
npm install @c2n/overlay
```

### Import

```typescript tag=CodeBlock
import '@c2n/overlay'
```

<br />
## Examples
<br />
---

```html tag=MdxCodeBlock,compact=true,component=c2-overlay
<div>
  <button popovertarget="demo-overlay">click me</button>
  <c2-overlay id="demo-overlay" popover="auto" client:only="lit">
    <div class="overlay-content">Overlay content</div>
  </c2-overlay>
</div>
<style>
  .overlay-content {
    background-color: #ffff;
    border: 1px solid #cccccc;
    padding: 24px;
    border-radius: 8px;
    user-select: none;
  }
</style>
```
